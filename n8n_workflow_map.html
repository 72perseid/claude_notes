n8n Workflow Architecture

Weekly RSS
Content Pipeline

Triggers every Monday · Reads 75 RSS feeds · Stores raw JSON in Google Sheets
Synthesizes via Claude · Outputs to any format on demand
01 · Trigger
⏰
Schedule Trigger
Every Monday 07:00 · Cron: 0 7 * * 1 · Fires the whole pipeline
n8n native
📋
Read RSS Sources Sheet
Google Sheets → ai_newsletters_v9 · Reads all 75 rows as JSON array · Extracts RSS Feed URL + Persona Tags columns
Google Sheets
🗓️
Set Week ID
Calculates current ISO week · Sets week_id = "2026-W10" · Used as sheet name + digest key downstream
Set node
02 · Fetch
🔁
Loop Over Feeds
SplitInBatches node · Iterates each of the 75 RSS feed URLs one by one · Passes persona_tags along with each URL
SplitInBatches
📡
Fetch RSS Feed
RSS Feed node · Parses XML → structured items · Returns: title, link, pubDate, content:encoded or description
RSS Feed node
🔍
Filter Last 7 Days
IF node · pubDate ≥ today minus 7 days · Drops anything older · Keeps only fresh items
IF node
🧹
Clean & Shape Item
Set node · Maps to standard schema: week_id, source, title, url, summary, published_at, persona_tags · Strips HTML from summary
Set node
// One cleaned item — this is the atom of the whole pipeline
{
  "week_id":      "2026-W10",
  "source":       "TechCrunch AI",
  "title":        "OpenAI launches new model for enterprise",
  "url":          "https://techcrunch.com/...",
  "summary":      "OpenAI announced a new model targeting...",
  "published_at": "2026-03-04T14:00:00Z",
  "persona_tags": "founder, enterprise"
}
03 · Store
➕
Create Weekly Sheet Tab
Google Sheets → Create Sheet · Name = week_id (e.g. "2026-W10") · Adds headers row automatically · Runs once per week
Google Sheets
📝
Append Rows to Weekly Tab
Google Sheets → Append · Writes every cleaned item as one row · All 75 feeds land here · Could be 200–400 rows/week
Google Sheets
💡
This sheet is your raw JSON source. Every downstream step reads from here. You can also open it directly in Sheets to browse, filter by persona_tags, or manually remove noise before synthesis runs.
04 · Synthesize
📖
Read Weekly Tab
Google Sheets → Get Rows · Reads the week's tab · Returns full JSON array of all items collected this week
Google Sheets
✂️
Condense for Claude
Code node · Keeps only: source, title, summary, persona_tags · Drops URL + dates · Reduces token count before API call
Code node
🤖
Claude — Master Digest
HTTP Request → Anthropic API · System prompt instructs Claude to output structured JSON · Returns themes, top launches, notable reads per persona
Claude API
// Claude's output — the master digest JSON (stored once, shaped many times)

  }

💾
Store Digest in Weekly Digests Sheet
Google Sheets → Append · One row per week · Columns: week_id, item_count, digest_json (stringified) · This is your permanent archive
Google Sheets
05 · Shape
⚡
This phase runs on demand, not on schedule. You trigger it manually or via webhook. It reads the stored digest JSON and reshapes it with a different Claude prompt each time. The digest never changes — only the prompt does.
🎯
Manual Trigger / Webhook
Input: week_id + format + persona · e.g. { week_id: "2026-W10", format: "linkedin", persona: "founder" }
On demand
📖

Google Sheets
